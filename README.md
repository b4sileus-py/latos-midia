# Latos Mídia

Landing page desenvolvida para a **Latos Mídia** como parte de uma atividade extensionista do curso de **Análise e Desenvolvimento de Sistemas da UniCesumar**.

## Sobre o projeto

A proposta do projeto foi criar uma página simples, responsiva e funcional para apresentar a marca, seus serviços, trabalhos realizados e canais de contato.

A Latos Mídia atua com:

- Identidade visual
- Captação e edição de fotos
- Captação e edição de vídeos
- Gerenciamento de redes sociais
- Criação de conteúdo para pequenos negócios e empresas locais

## Funcionalidades

- Layout responsivo para computador, tablet e celular
- Apresentação da marca e dos serviços
- Portfólio dividido por categorias
- Visualização ampliada das imagens
- Botão de contato direto pelo WhatsApp
- Link para o Instagram
- Destaque automático da seção atual no menu
- Botão para voltar ao topo
- Animações discretas durante a navegação
- Recursos básicos de acessibilidade

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages
